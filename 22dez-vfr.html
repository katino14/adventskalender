Eine

Metaplankartenabfrage

ohne Papier, in lesbarer Schrift, schnell und unkompliziert? Das geht mit oncoo.de

Das Tool ist ohne Registrierung oder Installation nutzbar. Es funktioniert auf allen Endgeräten (PC, Laptop, Tablet, Smartphone) und ist sehr intuitiv. Die Lehrkraft ruft die Webseite auf und erstellt durch Klicken auf den gelben Kreis ein neues oncoo. Daraufhin wird ein einmaliger Code erstellt, der den Lernenden zur Verfügung gestellt wird. Es gibt einige Einstellmöglichkeiten, um die Kartenabgabe zu steuern. Das fertige "Tafelbild" kann gesichert und über eine Lernplattform den Lernenden zur Verfügung gestellt werden. Wer es möchte, kann zusätzlich ein Passwort vergeben, um die Tafelansicht abzusichern.
